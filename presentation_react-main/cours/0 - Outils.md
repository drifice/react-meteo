## Lancer un projet en React (en utilisant le CLI)

```bash
npx create-react-app <nom_projet>
```

équivaut à (à ne pas exécuter!):

````bash
npm i -g create-react-app
create-react-app <nom_projet>
npm uninstall de create-react-app
````
